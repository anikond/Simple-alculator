# Лабораторная работа: Простой калькулятор

## Описание
Программа реализует простой калькулятор с возможностью ввода двух чисел и выполнения операций: сложение, вычитание, умножение, деление.

## Функции
- Ввод двух чисел.
- Операции:
  - Сложение
  - Вычитание
  - Умножение
  - Деление
- Обработка ошибок:
  - Деление на ноль
  - Ввод некорректных данных
- Кнопки:
  - Сброс — очистка всех полей
  - Закрыть — выход из программы

## Использование
1. Ввести два числа.
2. Нажать кнопку с нужной операцией.
3. Результат появится в отдельном поле.
4. Для очистки — нажать **Сброс**.
5. Для выхода — нажать **Закрыть**.
